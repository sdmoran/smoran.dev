Software Engineering

CS3733, WPI

Github Link (REPO CURRENTLY PRIVATE)

This project was completed as part of my Software Engineering course. Alongside 3 teammates, I developed a website to schedule and manage meetings, somewhat like When2Meet. We had functionality for a meeting organizer to input their availability and send a code to potential participants, who could then schedule meetings. Schedules can be modified or extended by organizers, or deleted altogether. We supported timeslots of varying lengths - 10, 15, 20, 30, or 60 minutes.

Backend developed in Java, with a MySQL database designed and manipulated by me. Frontend developed with HTML/CSS and Javascript, largely by my teammates. Used Amazon Web Serves Lambda, RDS, VPC, and API Gateway to connect backend to frontend.

Database Systems

CS3431, WPI

Github Link

For my databases course, I worked with a classmate to design a database meant to represent elements of a hospital. It included tables for doctors, patients, visits, rooms, pieces of equipment, and more. We planned and diagrammed the entire database, then constructed it, performed data entry, and finally created views and triggers.

The final product was an Oracle database, manipulated with SQL/PSQL as well as through JDBC with Java.
